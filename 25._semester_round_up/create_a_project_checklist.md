# Create a Project Checklist

**Type**: Group work

**Motivation**: With the exam approaching, it is important to have a clear picture of where your project stands and what is still missing.

**Exam**: Should be used for exam project

---

## Analyse your project

Let your LLM read through all files from [this semesters teachings](../README.md) (e.g. README, exercises, learning goals). Then let it create a checklist with all the elements we have covered.

Once the checklist is in place, go through your own Cookbook repository and mark each item as done or missing.

Add the missing elements to your Kanban board's backlog and prioritise them.

The missing parts you should fokus on doing before the exam period starts.
